### Installation

- Just, Clone this repository -
```
$ git clone git://github.com/CyberTitus/GPhisher.git
```

- Change to cloned directory and run `main.sh` -
```
$ cd GPhisher
$ bash main.sh
```

### Dependencies

**`GPhisher`** requires following programs to run properly - 
- `php`
- `wget`
- `curl`
- `git`
